### 💡 沉淀的核心方法论

- **AI 互评法（Cross-Model Evaluation）**：豆包出货 + DeepSeek 评分 + 人工拍板，0 成本完成多模型交叉验证
- **评分 Prompt 验收对齐原则**：评分标准必须与被测任务的约束一一对应
- **目标读者前置法（Target Audience First）**：先问"给谁看"再选最优版本
- **履历真实性边界**：能装专业，不能装履历
